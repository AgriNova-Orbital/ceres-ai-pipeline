本目錄用於整理 `Ceres AI / 穀神星AI` 的品牌素材，供 Google OAuth Brand Verification 與 OAuth 同意畫面使用。

## Required Files (for verification package)

請至少準備以下檔案：

1. `ceres-ai-logo.png` (primary app logo)
2. `ceres-ai-logo-120.png` (OAuth consent small logo)
3. `ceres-ai-logo-512.png` (high-resolution square logo)
4. `ceres-ai-wordmark.png` (horizontal wordmark for supporting materials)

## Recommended Specs

- **Format:** PNG (RGB, non-interlaced), transparent or solid background
- **Shape:** Square logo for consent screen and app identity
- **Minimum size:** 120 x 120 px
- **Recommended size:** 512 x 512 px
- **File size:** keep under 1 MB per file for faster upload/review
- **Visual consistency:** use the same brand name across assets: `Ceres AI / 穀神星AI`
- **No misleading marks:** do not include Google logos or partner badges in app logo
