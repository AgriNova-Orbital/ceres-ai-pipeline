# Logo Assets

Place brand assets here for frontend usage:

- `logo.png` (horizontal wordmark, recommended transparent PNG, ~1200x320)
- `logo-square.png` (square icon, recommended 512x512)
- `favicon.png` (recommended 64x64 or 128x128)

Current app references:

- `/logo/logo.png` on homepage hero
- `/logo/logo-square.png` on homepage header
- `/logo/favicon.png` in Next.js metadata icons

You can copy from repo root branding folder:

```bash
cp branding/logo.png frontend/public/logo/logo.png
cp branding/logo-square.png frontend/public/logo/logo-square.png
cp branding/favicon.png frontend/public/logo/favicon.png
```
